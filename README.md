# sosu-direct

It works on Windows only *(as of writing, 25/08/2024)*

## Usage

- `sosu://<beatmapset_id>` in Run *(WIN+R)*

## Requirements

- [Bun](https://bun.sh/) for `install.bat` that uses Bun as backend for downloading songs.

## Installation

- Run `install.bat`
- Done, use `sosu://<beatmapset_id>` to download.
